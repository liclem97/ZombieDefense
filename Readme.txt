[개발환경]
Windows, 언리얼엔진 4.18, VS2017

[빌드]
언리얼엔진 5.4, VS2022

[언어]
블루프린트

2인 플레이,  3인칭 슈팅게임
몰려오는 좀비로부터 넥서스를 보호해야합니다.
총 10웨이브로 구성되어있으며 마지막 웨이브에서 보스몬스터가 등장합니다.
좀비를 처치하여 얻을 수 있는 달러로 방어구, 탄창, 공격력 업, 방어력 업,  플래시라이트 등을 구매할 수 있습니다.

WindowServer 폴더 내의 데디케이트 서버를 열고,
데디케이트 서버를 실행한 컴퓨터의 Public IP와 포트번호(7777)를 입력 후 서버접속을 합니다. (ex. 127.0.0.1:7777)

게임 실행 후 방 생성 혹은 참가에 성공할 시 게임 맵으로 이동되며,
인원의 수가 만족되면 게임이 시작됩니다. 

[키 설명]
W,A,S,D : 이동
Spacebar : 점프
F : 손전등
X : 무기 장착
B : 상점
Tab : 캐릭터 정보
LMB : 발사
RMB : 조준

[게임실행]
Server : WindowsServer > ZombieDefenseServer.exe - 바로 가기
Client : WindowsClient > ZombieDefenseClient.exe - 바로 가기
